Формат файла

Excel файл должен содержать следующие колонки:

  • name - Название запчасти (обязательно)
  • manufacturer - Производитель (обязательно)
  • category - Категория (oil, filter, electrical, engine, transmission, suspension, other)
  • expected_lifespan_km - Ожидаемый ресурс в км (необязательно)
  • expected_lifespan_months - Ожидаемый ресурс в месяцах (необязательно)
  • unit - Единица измерения (pcs, liters, kg, set, meters)
  • current_price - Текущая цена за единицу
