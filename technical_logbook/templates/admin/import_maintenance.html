Формат файла

Excel файл должен содержать следующие колонки:

  • name - Название операции (обязательно)
  • system - Система автомобиля (обязательно)
  • car_model - Модель авто (необязательно, оставьте пустым для универсального регламента)
  • interval_km - Интервал в км (необязательно)
  • interval_months - Интервал в месяцах (необязательно)
  • labor_cost - Стоимость работы в рублях (необязательно)
